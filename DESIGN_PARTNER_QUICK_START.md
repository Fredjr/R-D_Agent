## âœ… Setup (5 minutes)

- [ ] Log in to R&D Agent
- [ ] Open or create a project
- [ ] Navigate to the **Questions** tab
- [ ] Add 3-5 papers to your project (if you haven't already)

---

## âœ… Create Your First Question (3 minutes)

- [ ] Click **"Add Main Question"** button
- [ ] Enter your main research question
- [ ] Select type: **Main**
- [ ] Set status: **Exploring**
- [ ] Set priority: **High** or **Medium**
- [ ] Click **"Create Question"**

**Example**: "How does sleep deprivation affect immune function?"

---

## âœ… Add Sub-Questions (3 minutes)

- [ ] Hover over your main question
- [ ] Click the **"+"** icon (Add Sub-Question)
- [ ] Enter a sub-question
- [ ] Set status and priority
- [ ] Click **"Create Question"**
- [ ] Repeat for 2-3 more sub-questions

**Examples**:
- "What immune markers are affected?"
- "What is the dose-response relationship?"
- "Are effects reversible?"

---

## âœ… Add a Hypothesis (2 minutes)

- [ ] Click on a question to expand it
- [ ] Scroll to **Hypotheses** section
- [ ] Click **"Add Hypothesis"** button
- [ ] Enter your hypothesis
- [ ] Select type: **Mechanistic**, **Predictive**, or **Descriptive**
- [ ] Set status: **Proposed** or **Testing**
- [ ] Set confidence: **Low**, **Medium**, or **High**
- [ ] Click **"Create Hypothesis"**

**Example**: "Sleep deprivation reduces NK cell activity through cortisol elevation"

---

## âœ… Link Evidence (2 minutes)

### Link Evidence to Question:
- [ ] Click on a question to expand it
- [ ] Click **"Link Evidence"** button
- [ ] Search for a paper
- [ ] Select the paper
- [ ] Choose evidence type: **Supports**, **Contradicts**, or **Neutral**
- [ ] Rate relevance: 1-10
- [ ] Add key finding (optional but recommended)
- [ ] Click **"Link"**

### Link Evidence to Hypothesis:
- [ ] Expand a question to see hypotheses
- [ ] Click **"Link Evidence"** on a hypothesis
- [ ] Search and select paper
- [ ] Choose evidence type: **Supports**, **Contradicts**, or **Neutral**
- [ ] Choose strength: **Weak**, **Moderate**, or **Strong**
- [ ] Add key finding
- [ ] Click **"Link"**

---

## âœ… Explore Features (Optional)

- [ ] Click evidence counts to view linked papers
- [ ] Edit a question (click pencil icon)
- [ ] Delete a question (click trash icon)
- [ ] Change question status
- [ ] Expand/collapse question branches
- [ ] Remove evidence (click X on evidence item)
